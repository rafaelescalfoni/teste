Questão 2

Escreva um programa que, dado um número inteiro digitado, informe se o número é primo ou não.

(Um método simples, porém pouco eficiente, é dividir o número por todos os números inteiros maiores que 1 até n-1 e verificar se o resto da divisão inteira é 0. Se sim, o número não é primo, caso contrário, tratase de um número primo).

Seu formulário deverá validar os dados de entrada, avisando caso a entrada seja inválida.
